### **9. LCM खोजें**

**पूर्ण कोड:**
```python
def find_lcm(x, y):
    greater = max(x, y)
    while True:
        if greater % x == 0 and greater % y == 0:
            lcm = greater
            break
        greater += 1
    return lcm

print(find_lcm(4, 6))
```

**व्याख्या:**
यह प्रोग्राम दो संख्याओं का न्यूनतम समापवर्त्य (LCM) खोजता है।

---

### **10. HCF खोजें**

**पूर्ण कोड:**
```python
def find_hcf(x, y):
    smaller = min(x, y)
    for i in range(1, smaller + 1):
        if x % i == 0 and y % i == 0:
            hcf = i
    return hcf

print(find_hcf(4, 6))
```

**व्याख्या:**
यह प्रोग्राम दो संख्याओं का उच्चतम समापवर्तक (HCF) खोजता है।
